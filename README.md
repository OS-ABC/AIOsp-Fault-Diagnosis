# AIOsp-Fault-Diagnosis

### 工具介绍

辅助运维人员进行微服务系统故障诊断及修复，故障根因诊断粒度为诊断到指标或日志，修复方案通过已有的图谱生成，用Flask实现查询接口，实现功能如下：
- 服务依赖图构建：还原出系统实时服务依赖关系并在图中标识出存在异常服务和可能的根因服务
- 异常数据提取与依赖图构建：对异常服务进一步分析，构建异常间的因果关系，在图中标识出根源指标或日志
- 异常日志根源辅助分析：基于日志匹配分析异常日志可能的原因及解决方案
